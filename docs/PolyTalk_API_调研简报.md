## 七、API 封装方案（MVP版）

### 架构设计
```
┌─────────────────────────────────────┐
│         PolyTalk MVP                │
├─────────────────────────────────────┤
│  FastAPI Router (统一入口)          │
│  ├─ /chat   → GLM-4-Flash           │
│  ├─ /stt    → Local Whisper         │
│  └─ /tts    → Edge TTS (可选)       │
├─────────────────────────────────────┤
│  SQLite (本地存储)                   │
│  ├─ conversations                   │
│  ├─ users                          │
│  └─ settings                       │
├─────────────────────────────────────┤
│  Services (核心服务)                 │
│  ├─ NLPService (GLM-4-Flash SDK)    │
│  ├─ STTService (faster-whisper)     │
│  └─ TTSService (edge-tts)           │
└─────────────────────────────────────┘
```

### 技术栈（MVP）
- **Language**: Python 3.11+
- **Framework**: FastAPI
- **Database**: SQLite + SQLAlchemy
- **NLP**: zhipuai (智谱GLM SDK)
- **STT**: faster-whisper (本地GPU/CPU)
- **TTS**: edge-tts (微软免费)

### 依赖清单
```txt
fastapi>=0.109.0
uvicorn>=0.27.0
sqlalchemy>=2.0
zhipuai>=2.0
faster-whisper>=1.0
edge-tts>=6.1
python-multipart
aiofiles
```

### 扩展预留
- 后期可切换到 Claude/火山引擎（适配器模式）
- SQLite → PostgreSQL（用户量增长后）
